# Accessibility prime directive

## Core philosophy

A1 stands for accessibility first and I take that very seriously.

Every single UI component in the A1 system is built from the ground up with functional semantics and structure that allows them to work for everyone.

- Web Content Accessibility Guidelines (referred to as WCAG) 2.2 AA compliance
- Web Accessibility Initiative - Accessible Rich Internet Applications Authoring Practices Guide (referred to as WAI-ARIA APG)

Native HTML first, role and ARIA attributes used carefully where they provide better experience for users

- https://www.w3.org/WAI/ARIA/apg/practices/read-me-first/
